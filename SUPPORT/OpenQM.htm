AE (Alternative Editor)

This is an emulation of the AE command found in the Unidata product but it includes several extensions added by its author, Keith Johnson.

To obtain a copy of this editor, right click here to download a WinZip file containing the QMBasic source code, an HTML help file and a set of message texts.

The messages can be added to the QM message library by executing a command
   LOAD.LANGUAGE AE_MESS
in the QMSYS account where AE_MESS is the pathname of the unzipped message file from this download.
